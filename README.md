# SSOO tester
## _Run your tests easily_

### Preparing environment

1. Copy the `testsT0/` folder into your main folder
2. Add `tester.sh` to your main folder

### Running Tests

**Run all tests**
```sh
sh tester.sh
```
**If you want to run specific test**
```sh
sh tester.sh <option>
```
Where `<option>` must be a number in {1...5}
